# Net Messenger (ASP.NET Web Api + Windows Forms App)

> Project made using [ASP.NET SignalR](https://dotnet.microsoft.com/en-us/apps/aspnet/signalr)

## How to run API service

> Microsoft SQL Server and Visual Studio must be installed!

1. Create database named **"NetMessenger"**.
2. Open **"Messenger.sln"**.
3. Start **"Messenger.DatabaseInitialization"** project and wait until the tables and stored procedures will be created.
4. Start **"Messenger.PresentationLogic"** project.


## How to run Windows Forms App

1. Start **"Messenger.App"** project.
